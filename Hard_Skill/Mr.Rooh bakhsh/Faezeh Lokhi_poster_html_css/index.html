که ماموریت اصلی خودمون رو ایجاد بسترهای لازم برای شکوفایی توانمندی‌های

منجر به رشد نوآوری، ترویج علم، آموزش و کاربردی‌سازی فناوری‌های پیشرفته در

سطح مدرسه، دانشگاه، صنعت و جامعه تعریف کردیم.
